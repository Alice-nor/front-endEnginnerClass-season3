* 網站上傳與維護 製作好的網站經測試可利用 FTP 軟體上傳到網站伺服器中 *


IP 轉換成 Domain Name，需要使用 網域名稱系統 DNS（Domain Name System）。網域名稱第一個為資訊種類（www、bbs、mail、FTP），第二個為主機名稱（seed、nctu、cvtc），後面為領域名稱團體（edu、org、com）及地區（tw、cn、jp）。
